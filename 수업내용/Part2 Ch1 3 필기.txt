어드민 기능 개요
- 일별 가입 유저수를 통해 메인 페이지의 그래프 대시보드 기능 구현
- 유저, 게시글 리스트 테이블 기능 구현

-> thymeleaf + bootstrap를 통한 기능 구현 -> MVC 패턴 적용

기본적으로 템플릿들의 설정과 컨트롤러, 그리고 뷰단에 타임리프 사용방법들을 실질적으로 확인 및 응용
+ HTML, CSS, JS 같은 수업자료들의 경우 공유하니까 참고할 것(템플릿만 알맞은 위치에 넣어주기)

어드민 기능
- 기본적인 CRUD 가 대부분인 Admin 기능을 효율적으로 빠르게 만드는 방법
  (통일된 응답값듵을 위한 기능, 페이징을 위한 기본적인 기능)
  (객체지향을 활용해 반복없이 구현 예정)
- 많은 데이터와 대시보드 기능 구현을 위한 대규모 데이터 조회방법
-> 인덱스 (+커버링 인덱스)를 통한 효율적인 페이징 기능 구현
